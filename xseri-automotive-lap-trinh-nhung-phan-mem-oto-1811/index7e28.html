C++ là ngôn ngữ phổ biến thứ 3 trên thế giới, chỉ sau Java và C (6/2018 )


1. Hiện nay ngôn ngữ C++ có được dùng phổ biến không?

Xem xếp hạng mức độ sử dụng của các ngôn ngữ lập trình trên trang 

2. Học C++ có làm được C không?

C++ = C và hướng đối tượng => lập trình tốt C++ sẽ dễ dàng chuyển sang lập trình C.

C++ chạy nhanh, kích thước chương trình nhỏ, thư viện lập trình mạnh phù hợp cho các máy tính trên ôtô (điều khiển động cơ, hệ thống an toàn, hệ thống thông tin giải trí,…

3. Sao lập trình ứng dụng ô tô lại dùng C++?

Cần có kỹ năng lập trình và kiến thức về cơ sở dữ liệu cơ bản như: hiểu, sử dụng được các khái niệm: biến, lặp, rẽ nhánh, hàm; viết được các chương trình lấy input từ bàn phím, xử lý theo logic của bài toán, đưa kết quả ra màn hình,... bằng bất kỳ ngôn ngữ nào. Có kỹ năng viết lệnh SQL để truy xuất cơ sở dữ liệu.

4. Cần nền tảng gì để học khóa này?

Nên học 3 môn của FUNiX: PRF192x (Creating your first program with Javascript – Xây dựng phần mềm đầu tiên), WEB101x (Building your first website - Xây dựng phần mềm đàu tiên) và DBI202x (Database Fundamentals – Các hệ cơ sở dữ liệu)

5. Nếu chưa bao giờ lập trình thì cần học môn nào trước khi học khóa này?

Khóa học trang bị kỹ năng lập trình C++ định hướng cho công nghiệp phần mềm trên ô tô nên nó là thực hành. Việc học chủ yếu thông qua làm các dự án nhỏ và dự án tốt nghiệp cuối khóa. Các dự án được đề xuất bởi FGA để giúp người học làm quen dần với thực tế công việc ngay trong quá trình học.
